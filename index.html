About Us

About the 27th MUBS Guild ICT Ministry

The 27th MUBS Guild ICT Ministry headed by Hon. Matovu Solomon H Kacungwa is dedicated to advancing the technological infrastructure of the university. Our mission is to improve digital services, ensure robust support for students and faculties, and drive innovation in technology use across campus. We are committed to providing top-notch ICT services, including technical support, digital resource management, and ensuring that all technological needs are met efficiently.

Purpose of the Platform

The MUBS ICT Ministry Service Centre is designed to provide comprehensive support and resources for students and staff at MUBS. Our platform aims to streamline access to essential services, facilitate seamless communication, and enhance the overall university experience. Whether you need assistance with e-learning, managing your student portal, or contacting the ICT support team, our platform is here to assist you efficiently and effectively.

Software Assistance

Our Software Support Services

At the MUBS ICT Ministry Service Centre, we are dedicated to providing students with the essential software tools required for academic and personal success. Our comprehensive software support services include:

  • Software Installation and Configuration:

    Assistance with installing and configuring software necessary for coursework, research, and other academic tasks.

  • Troubleshooting and Support:

    Help with resolving issues related to software functionality, compatibility, and updates.

  • Training and Tutorials:

    Offering training sessions and tutorials to help students effectively use software tools and applications.

  • Software Recommendations:

    Providing recommendations for software solutions tailored to academic requirements and personal needs.

How to Access Software Assistance

To receive software assistance, students can:

  • Visit the ICT Ministry Office:

    Receive in-person support and consultations at our office.

  • Submit a Request Online:

    Complete our online support request form available on our website for assistance with software issues.

  • Attend Workshops:

    Join our workshops and training sessions to learn about various software tools.

  • Contact Us via WhatsApp:

    Reach out to our support team directly through WhatsApp for immediate assistance.

We are committed to equipping students with the resources necessary for academic excellence. Our team is here to address any software-related concerns you may have.

Keyboard Shortcuts

Enhance your productivity with these essential keyboard shortcuts:

  • Ctrl + C: Copy selected text or item.
  • Ctrl + V: Paste copied text or item.
  • Ctrl + X: Cut selected text or item.
  • Ctrl + Z: Undo the last action.
  • Ctrl + Y: Redo the last undone action.
  • Ctrl + A: Select all text or items in a document or window.
  • Alt + Tab: Switch between open applications.
  • Ctrl + S: Save your current document or project.
  • Ctrl + P: Print the current document or page.
  • Ctrl + F: Find text within a document.
  • Ctrl + H: Find and replace text within a document.
  • Ctrl + N: Open a new document or window.
  • Ctrl + O: Open an existing document or file.
  • Ctrl + T: Open a new tab in a web browser.
  • Ctrl + W: Close the current document or tab.
  • Ctrl + Q: Quit the current application.
  • Ctrl + D: Bookmark the current webpage in a browser.
  • Ctrl + L: Highlight the address bar in a web browser.
  • Ctrl + Shift + T: Reopen the last closed tab in a web browser.
  • Alt + F4: Close the current window or application.
  • F2: Rename the selected item or file.
  • F5: Refresh the current webpage or document.
  • Ctrl + Esc: Open the Start menu.
  • Windows Key + D: Show or hide the desktop.
  • Windows Key + L: Lock your computer.
  • Ctrl + Shift + Esc: Open the Task Manager.
  • Windows Key + E: Open File Explorer.
  • Windows Key + R: Open the Run dialog box.
  • Windows Key + P: Project your screen to a connected display.
  • Windows Key + I: Open the Settings window.
